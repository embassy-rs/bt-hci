# bt-hci-serial

HCI transport implementation for a split serial bus using the [UART transport layer protocol](https://www.bluetooth.com/wp-content/uploads/Files/Specification/HTML/Core-54/out/en/host-controller-interface/uart-transport-layer.html),
implementing the [`bt-hci-transport`](https://crates.io/crates/bt-hci-transport) `Transport` trait.
